Swim Database

REACT | REDUX | AMAZON S3 | SESSIONS | AUTH0 | MATERIAL-UI

The app is used to store swimming event and race information. This app was built using react. The interface was developed using material-ui. It uses auth0 and sessions to authenticate and register users and allow them to access and interact with their data. Amazon S3 is used to store images used throughout the app. Redux is used for storing data used for intercomponent communication.
